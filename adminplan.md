CMJ ADMIN PANEL - PROPOSAL

Overview:
Admin dashboard to manage the CMJ Gold & Diamond Jewellers website, including product management and content control.

Features:

1. Admin Login
   - Password protected access
   - Secure session management

2. Dashboard
   - Product statistics overview
   - Quick action buttons

3. Product Management
   - View all products in table/grid
   - Add new products (name, image, price, weight, category)
   - Edit existing products
   - Delete products

Technical Details:
- Built with Next.js
- API routes for data operations
- Real-time updates

Estimated Timeline: ~1 hour development
